# iug-study-buddy

## Project Description
A responsive single-page web application built with HTML, CSS, and vanilla JavaScript. It helps students manage tasks, track weekly study habits, and organize learning resources through a clean dashboard interface.

## Features
- SPA navigation with dashboard, tasks, habits, resources, and settings
- Tasks CRUD with filters, sorting, validation, and localStorage persistence
- Weekly habit tracker with goal progress and streak summaries
- Resource library loaded via fetch, with search, filters, and favorites
- Theme toggle and reset data controls

## How to Run
1. Start a local server (recommended for fetch to work).
   - `python -m http.server`
2. Open `http://localhost:8000` in your browser.
3. Or use VS Code Live Server.
